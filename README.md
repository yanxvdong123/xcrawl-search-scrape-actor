# XCrawl Web Search & Scrape

An Apify Actor that searches the web and scrapes pages using the XCrawl proxy network.

**Features:** web search, page scraping, JS rendering, concurrent crawling, anti-block, global IPs.

See [github.com/yanxvdong123/xcrawl-search-scrape-actor](https://github.com/yanxvdong123/xcrawl-search-scrape-actor) for full docs.
